# Command Line Interface

## Install
```
go get github.com/inabajunmr/kotrcli/cli/kot
```

## Config
You need `~/.kot/config` and put data as follow format. You need get token from Web UI.

```
[default]
user_token = xxxxxx
token = xxxxxx
```

## Usage
```
$ kot syukkin
$ kot taikin
```
